- 设置团队或小组成员 业务流程
```mermaid
flowchart LR
开始 -->T[权限校验] -->A[获取团队所有旧成员并移除] -->B[记录新团队成员Ids] --> C{判断新团队成员是否为空}
C -->|是| C0[直接移除团队所有成员] -->结束
C -->|否| D[清理旧成员] -->E[插入新成员]
 -->结束
```
